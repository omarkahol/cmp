The cross-\/correlation $R_{ij}(\tau)$ between feature $i$ and $j$ can be computed in the frequency domain\+:   \[R_{ij}(\tau) = \mathcal{F}^{-1}\left\{ \mathcal{F}\{x_i\} \cdot \mathcal{F}\{x_j\}^* \right\}
