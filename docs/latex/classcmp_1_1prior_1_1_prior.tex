\doxysubsection{Detailed Description}
Base class for prior probability distributions. 

{\bfseries{Mathematical}} {\bfseries{Formulation}} Represents a log-\/prior probability density function $ \log p(\theta) $ over parameters $ \theta \in \mathbb{R}^d $. Provides virtual interfaces for evaluation, gradient, and Hessian computation\+:
\begin{DoxyItemize}
\item Value\+: $ f(\theta) = \log p(\theta) $
\item Gradient\+: $ \nabla_i f(\theta) = \frac{\partial \log p(\theta)}{\partial \theta_i} $
\item Hessian\+: $ \mathcal{H}_{ij} f(\theta) = \frac{\partial^2 \log p(\theta)}{\partial \theta_i \partial \theta_j} $
\end{DoxyItemize}

{\bfseries{Implementation}} {\bfseries{Algorithm}} Pure virtual interface specifying the contract for prior evaluation. Concrete subclasses must implement {\ttfamily eval}, {\ttfamily eval\+Gradient}, and {\ttfamily eval\+Hessian}.
