\doxysubsection{Detailed Description}
Abstract base class for Gaussian Process prior mean functions. 

Mathematical Formulation The mean function $m: \mathbb{R}^D \to \mathbb{R}$ defines the expected value of the GP prior at any input $\mathbf{x}$\+:   \[m(\mathbf{x}) = \mathbb{E}[f(\mathbf{x})]
\] Commonly configured as a constant $m(\mathbf{x}) = c$ or zero $m(\mathbf{x}) = 0$.

Implementation Algorithm Provides virtual interfaces for mean evaluations ({\ttfamily eval}), first-\/order gradients ({\ttfamily eval\+Gradient}) with respect to a parameter $p_i$, and second-\/order Hessians ({\ttfamily eval\+Hessian}).
