CMP++: Scientific Uncertainty Quantification and Bayesian Calibration Library

Introduction

Welcome to the official API documentation for CMP++, a high-performance, template-driven C++ library specifically engineered for Uncertainty Quantification (UQ), Bayesian Calibration, and Machine Learning in complex scientific computing environments.

Architectural & Mathematical Overview

CMP++ solves the classical Bayesian calibration formulation for computer models:

\[ y(\mathbf{x}) = \eta(\mathbf{x}, \boldsymbol{\theta}) + \delta(\mathbf{x}) + \varepsilon \]

where:

  • \(\eta(\mathbf{x}, \boldsymbol{\theta})\) is the deterministic computer simulator parameterized by calibration inputs \(\boldsymbol{\theta}\).
  • \(\delta(\mathbf{x})\) is the systematic model discrepancy, represented non-parametrically as a Gaussian Process: \(\delta(\mathbf{x}) \sim \mathcal{GP}(0, k(\mathbf{x}, \mathbf{x}'))\).
  • \(\varepsilon \sim \mathcal{N}(0, \sigma^2)\) is the experimental observation noise.

Quick-Start & Core Headers
