\subsubsection*{template$<$typename Basis\+Type$>$\newline
class cmp\+::\+Tensor\+Integrator$<$ Basis\+Type $>$}
Multi-\/dimensional tensor product quadrature integrator. 

Mathematical Formulation\+: A multi-\/dimensional integral over a tensor-\/product domain \$\textbackslash{}\+Omega = \mbox{[}a\+\_\+1, b\+\_\+1\mbox{]} \textbackslash{}times \textbackslash{}dots \textbackslash{}times \mbox{[}a\+\_\+d, b\+\_\+d\mbox{]}\$ is approximated by the tensor product of 1D quadrature rules\+:   \[\int_{\Omega} f(\mathbf{x}) \, d\mathbf{x} \approx \sum_{i_1=0}^{N-1} \dots \sum_{i_d=0}^{N-1} \left( \prod_{j=1}^{d} w_{i_j} \right) f(x_{i_1}, \dots, x_{i_d})
\] where \$w\+\_\+\{i\+\_\+j\}\$ and \$x\+\_\+\{i\+\_\+j\}\$ are the 1D quadrature weights and nodes along dimension \$j\$. For physical domain integration over custom bounds \$\mbox{[}p\+\_\+\{1,d\}, p\+\_\+\{2,d\}\mbox{]}\$, a coordinate transformation maps the canonical nodes \$\textbackslash{}xi \textbackslash{}in \mbox{[}-\/1, 1\mbox{]}\$ (or the canonical domain of the basis) to \$x \textbackslash{}in \mbox{[}p\+\_\+\{1,d\}, p\+\_\+\{2,d\}\mbox{]}\$ via\+:   \[x = \text{mapToPhysical}(\xi, p_{1,d}, p_{2,d})
\] The integration automatically incorporates any scaling factors (Jacobian of the transformation) if required by the basis.

Implementation Algorithm\+:
\begin{DoxyEnumerate}
\item Instantiates a 1D quadrature rule {\ttfamily \doxylink{classcmp_1_1_quadrature1_d}{Quadrature1D}\texorpdfstring{$<$}{<}Basis\+Type\texorpdfstring{$>$}{>}} with the specified {\ttfamily points\+Per\+Dim}.
\item Allocates a grid node matrix of size \$N\texorpdfstring{$^\wedge$}{\string^}d \textbackslash{}times d\$ and a weight vector of size \$N\texorpdfstring{$^\wedge$}{\string^}d\$, where \$\+N\$ is {\ttfamily points\+Per\+Dim} and \$d\$ is {\ttfamily dim}.
\item Iterates through all \$N\texorpdfstring{$^\wedge$}{\string^}d\$ combinations using base-\/\$\+N\$ digit decomposition to reconstruct the index tuple \$(i\+\_\+0, \textbackslash{}dots, i\+\_\+\{d-\/1\})\$.
\item For each joint grid point \$i\$, assigns the \$d\$-\/th dimension coordinate to the corresponding 1D node, and multiplies the 1D weights to form the joint weight \$\+W\+\_\+i = \textbackslash{}prod\+\_\+\{j=0\}\texorpdfstring{$^\wedge$}{\string^}\{d-\/1\} w\+\_\+\{i\+\_\+j\}\$.
\item Provides two {\ttfamily integrate} overloads\+:
\begin{DoxyItemize}
\item Canonical {\ttfamily integrate(f)}\+: computes \$\textbackslash{}sum\+\_\+\{i\} W\+\_\+i f(\textbackslash{}mathbf\{x\}\+\_\+i)\$.
\item Physical {\ttfamily integrate(f, p1, p2)}\+: maps canonical nodes to physical coordinates using {\ttfamily Basis\+Type\+::map\+To\+Physical} for each dimension before evaluating \$f\$ and summing with weights. 
\end{DoxyItemize}
\end{DoxyEnumerate}
