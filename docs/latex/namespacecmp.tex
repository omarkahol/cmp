double cmp\+::fd\+\_\+gradient (\begin{DoxyParamCaption}\item[{const Eigen\+::\+Vector\+Xd \&}]{x\+\_\+0}{, }\item[{const std\+::function$<$ double(const Eigen\+::\+Vector\+Xd \&)$>$}]{fun}{, }\item[{const int \&}]{i}{, }\item[{const \mbox{\hyperlink{namespacecmp_a7cdd4e585f27736074b06674fdab41cc}{accuracy}}}]{order}{ = {\ttfamily \mbox{\hyperlink{namespacecmp_a7cdd4e585f27736074b06674fdab41cca697b45a18e13243e10fbc8575ea08d69}{SECOND}}}, }\item[{const double}]{h}{ = {\ttfamily 1E-\/5}}\end{DoxyParamCaption})}



Computes the i-\/th component of the gradient of the function fun evaluated at x\+\_\+0 using a central difference scheme. 

Mathematical Formulation Approximates the partial derivative of $f: \mathbb{R}^D \to \mathbb{R}$ at $\mathbf{x}_0$ with respect to the $i$-\/th variable using a central difference stencil\+:   \[\frac{\partial f(\mathbf{x}_0)}{\partial x_i} \approx \frac{f(\mathbf{x}_0 + h \mathbf{e}_i) - f(\mathbf{x}_0 - h \mathbf{e}_i)}{2h}
\] where $\mathbf{e}_i$ is the $i$-\/th standard basis vector, and $h > 0$ is the step size. For higher-\/order accuracy (e.\+g., 4th, 6th, or 8th order), wider stencils are employed.

Implementation Algorithm
\begin{DoxyEnumerate}
\item Creates coordinate perturbations $\mathbf{x}_0 \pm n h \mathbf{e}_i$ for stencil points $n$.
\item Evaluates the function {\ttfamily fun} at each perturbed point.
\item Takes a linear combination of the function values scaled by the central difference stencil coefficients.
\end{DoxyEnumerate}
