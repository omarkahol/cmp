\doxysubsection{Detailed Description}
Isotropic bandwidth matrix (single scalar parameter h). 

Mathematical Formulation Defines a spherical scaling matrix $\mathbf{H} = h \mathbf{I}_D$. The transformation scales inputs uniformly\+:   \[\mathbf{z} = \frac{1}{h} (\mathbf{x}_1 - \mathbf{x}_2)
\] The determinant is $\det(\mathbf{H}) = h^D$.
