\doxysubsection{Detailed Description}
Represents a \doxylink{classcmp_1_1covariance_1_1_linear}{Linear} covariance function. 

Mathematical Formulation If {\ttfamily index\+X\+\_\+} is -\/1, computes the full dot product\+:   \[k_{\text{lin}}(\mathbf{x}_1, \mathbf{x}_2) = \mathbf{x}_1^T \mathbf{x}_2
\] Otherwise, evaluates only for the coordinate at index\+X\+\_\+\+:   \[k_{\text{lin}}(\mathbf{x}_1, \mathbf{x}_2) = x_{1,\text{index}} \times x_{2,\text{index}}
\]
