\doxysubsection{Detailed Description}
Blended mean function that interpolates local GP means using classifier probabilities. 

Mathematical Formulation The blended mean function at input \$x\$ is evaluated as\+:   \[M(x) = \sum_{k=1}^K P(C=k | x) m_k(x; \theta_k)
\] where \$P(C=k \texorpdfstring{$\vert$}{|} x)\$ is the classifier-\/derived probability that \$x\$ belongs to cluster \$k\$, and \$m\+\_\+k\$ is the mean function of the \$k\$-\/th cluster\textquotesingle{}s GP.

Implementation Algorithm
\begin{DoxyEnumerate}
\item Queries the classifier for the cluster membership probability vector at input \$x\$.
\item Evaluates the local mean function for each cluster GP at \$x\$.
\item Returns the dot product of the probability vector and the local GP mean vector. 
\end{DoxyEnumerate}
