Detailed Description

Abstract base class for all classifiers.

Mathematical Formulation A classifier maps an input vector \(\mathbf{x} \in \mathbb{R}^D\) to a class label \(y \in \{0, 1, \dots, C-1\}\), or estimates the class posterior probabilities \(P(y = c \mid \mathbf{x})\) for \(c = 0, \dots, C-1\).

Implementation Algorithm Provides a pure virtual interface defining:

  • predictProbabilities(x): Returns a vector of probability estimates.
  • predict(x): Returns the class label with the highest probability.
