\doxysubsection{Detailed Description}
Manages a clustered set of Gaussian Processes for localized regression. 

\$.

For a query point \$x\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*}\$, a classifier predicts the probability of cluster membership \$P(C = k \textbackslash{}mid x\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*})\$. The blended prediction is given by\+:   \[\mu(x^*) = \sum_{k=1}^K P(C = k \mid x^*) \mu_k(x^*)
\]   \[\sigma^2(x^*) = \sum_{k=1}^K P(C = k \mid x^*)^2 \sigma_k^2(x^*)
\] where \$\textbackslash{}mu\+\_\+k(x\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*})\$ and \$\textbackslash{}sigma\+\_\+k\texorpdfstring{$^\wedge$}{\string^}2(x\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*})\$ are the mean and variance predictions of the \$k\$-\/th GP.

Switching steps evaluate the change in predictive log-\/likelihood (LOO error for current cluster, or standard predictive log-\/likelihood for candidate clusters). The stochastic switching probability for point \$i\$ to cluster \$j\$ is computed as\+:   \[p_{ij} \propto P(C = j \mid x_i) \exp\left( \frac{\Delta_{ij}}{T} \right)
\] where \$\textbackslash{}\+Delta\+\_\+\{ij\}\$ represents the log-\/predictive score difference if point \$i\$ is moved to cluster \$j\$, and \$\+T\$ is the temperature parameter.

Implementation Algorithm
\begin{DoxyEnumerate}
\item {\bfseries{Initialization ({\ttfamily condition})}}\+: Groups the input observations according to their labels, instantiates the GP models, computes initial cluster centroids, and sets up index lookup tables.
\item {\bfseries{Blended Prediction ({\ttfamily predict})}}\+: Obtains probabilities from the classifier, queries predictions from each local GP, and performs the weighted combination.
\item {\bfseries{Stochastic Switching ({\ttfamily switch\+Step})}}\+: Precomputes probability distributions for all points based on likelihood shifts, filters active points, and samples switches up to {\ttfamily max\+Allowed\+Switches}.
\item {\bfseries{Deterministic Switching ({\ttfamily deterministic\+Switch\+Step})}}\+: Calculates the log-\/likelihood gain for all possible re-\/assignments, sorts them, and executes the top beneficial moves.
\item {\bfseries{Purging / Merging}}\+: Reassigns points from a dissolved cluster to their nearest centroids, shrinks the list of active GPs, and shifts all cluster labels to maintain contiguous indices.

Mathematical Formulation Let the dataset be composed of \$\+N\$ observations \$(x\+\_\+i, y\+\_\+i) \textbackslash{}in \textbackslash{}mathbb\{R\}\texorpdfstring{$^\wedge$}{\string^}d \textbackslash{}times \textbackslash{}mathbb\{R\}\$. The data is partitioned into \$\+K\$ clusters. Each cluster \$k \textbackslash{}in 1, \textbackslash{}ldots, K 
\end{DoxyEnumerate}
