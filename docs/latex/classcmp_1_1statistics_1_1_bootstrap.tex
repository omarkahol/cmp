\doxysubsection{Detailed Description}
\doxylink{classcmp_1_1statistics_1_1_bootstrap}{Bootstrap} resampling index generator. 


\begin{DoxyItemize}
\item 
\end{DoxyItemize}\hypertarget{classcmp_1_1statistics_1_1_bootstrap_autotoc_md25}{}\doxysubsubsubsection{\texorpdfstring{Mathematical Formulation}{Mathematical Formulation}}\label{classcmp_1_1statistics_1_1_bootstrap_autotoc_md25}
Draws a random sample $\mathcal{D}^*$ of size $M$ from a dataset $\mathcal{D}$ of size $N$.
\begin{DoxyItemize}
\item If with replacement\+:   \[P(x_i^* = x_j) = \frac{1}{N} \quad \forall i \in \{1,\dots,M\}, \ j \in \{1,\dots,N\}
\]
\item If without replacement\+:   \[P(x_i^* = x_j) = \frac{1}{N - i + 1}
\] subject to all selections being distinct.
\item \#\#\# Implementation Algorithm
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item If {\ttfamily with\+Replacement\+\_\+} is true, generates $M$ random integers uniformly distributed in the range $[0, N-1]$.
\item If false, shuffles the index array using {\ttfamily std\+::shuffle} and extracts the first $M$ elements. 
\end{DoxyEnumerate}
