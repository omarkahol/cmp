\doxysubsection{Detailed Description}
Represents an \doxylink{classcmp_1_1covariance_1_1_inverse}{Inverse} covariance function. 

Mathematical Formulation If {\ttfamily index\+X\+\_\+} is -\/1, computes\+:   \[k_{\text{inv}}(\mathbf{x}_1, \mathbf{x}_2) = \frac{1}{\mathbf{x}_1^T \mathbf{x}_2 + 1}
\] Otherwise, evaluates only for the coordinate at index\+X\+\_\+\+:   \[k_{\text{inv}}(\mathbf{x}_1, \mathbf{x}_2) = \frac{1}{x_{1,\text{index}} \times x_{2,\text{index}} + 1}
\]
