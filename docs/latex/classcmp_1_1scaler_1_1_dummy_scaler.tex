\doxysubsection{Detailed Description}
A pass-\/through scaler that leaves input data unchanged. 


\begin{DoxyItemize}
\item 
\end{DoxyItemize}\hypertarget{classcmp_1_1scaler_1_1_dummy_scaler_autotoc_md18}{}\doxysubsubsubsection{\texorpdfstring{Mathematical Formulation}{Mathematical Formulation}}\label{classcmp_1_1scaler_1_1_dummy_scaler_autotoc_md18}
The transformation is the identity function mapping $ \mathbb{R}^d \to \mathbb{R}^d $\+: \[ T(x) = x \] \[ T^{-1}(y) = y \] The intercept $ \mu $ is the zero vector, and the scale $ S $ is the identity matrix $ I $.
