Detailed Description

Represents a constant scale covariance function.

Mathematical Formulation For a constant kernel, the covariance is independent of the input points:

\[k_{\text{const}}(\mathbf{x}_1, \mathbf{x}_2; \boldsymbol{\theta}) = \theta_{\text{index}}^2 \]
