\doxysubsection{Detailed Description}
Linearly scales features to a target bounding box range. 

Mathematical Formulation Scales each feature vector component to live within range $[a_j, b_j]$ (typically $[0, 1]$)\+:   \[y_j = a_j + \frac{x_j - x_{\text{min}, j}}{x_{\text{max}, j} - x_{\text{min}, j}} (b_j - a_j)
\] where $x_{\text{min}, j}$ and $x_{\text{max}, j}$ are the training data\textquotesingle{}s minimum and maximum values for feature $j$.

Implementation Algorithm
\begin{DoxyEnumerate}
\item {\ttfamily \doxylink{classcmp_1_1scaler_1_1_min_max_scaler_a16823cc81232127bc86759957d1c7cd5}{fit()}} computes column-\/wise minima and maxima vectors of the training matrix.
\item {\ttfamily \doxylink{classcmp_1_1scaler_1_1_min_max_scaler_a566ca9d333c4703307bd78e7afaebf0c}{transform()}} and {\ttfamily \doxylink{classcmp_1_1scaler_1_1_min_max_scaler_a23768d4063032ff713f6cdd9092240b4}{inverse\+Transform()}} apply element-\/wise division and scaling multiplication. 
\end{DoxyEnumerate}
