\doxysubsection{Detailed Description}
Multivariate Kernel Density Estimation (\doxylink{classcmp_1_1density_1_1_k_d_e}{KDE}) class. 

Mathematical Formulation Evaluates the multivariate probability density function estimate of a random vector $\mathbf{x} \in \mathbb{R}^D$ based on observations $\{\mathbf{x}_i\}_{i=1}^N$\+:   \[\hat{f}(\mathbf{x}) = \frac{1}{N} \sum_{i=1}^N \frac{1}{\det(\mathbf{H})} K\left( \mathbf{H}^{-1} (\mathbf{x} - \mathbf{x}_i) \right)
\] where $\mathbf{H}$ is the bandwidth matrix and $K$ is the multi-\/dimensional kernel function.
\begin{DoxyItemize}
\item Scott\textquotesingle{}s Rule\+: $H_{dd} = N^{-1/(D+4)} \hat{\sigma}_d$
\item Silverman\textquotesingle{}s Rule\+: $H_{dd} = \left(\frac{4}{D+2}\right)^{1/(D+4)} N^{-1/(D+4)} \hat{\sigma}_d$ where $\hat{\sigma}_d$ is the standard deviation of coordinate $d$.
\end{DoxyItemize}

Implementation Algorithm
\begin{DoxyEnumerate}
\item {\ttfamily \doxylink{classcmp_1_1density_1_1_k_d_e_a9c351627adef2b24a76e8c53bbcea22b}{condition()}} registers the dataset references.
\item {\ttfamily \doxylink{classcmp_1_1density_1_1_k_d_e_a0de562b368818bb4de7a7d28d5e7ff4b}{eval()}} averages kernel evaluations over all conditioning observation points, scaled by the determinant of the bandwidth matrix. 
\end{DoxyEnumerate}
