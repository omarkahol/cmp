\doxysubsection{Detailed Description}
Simple partitioning algorithm that assigns observations to clusters uniformly at random. 

Mathematical Formulation Each sample index $i$ is assigned a cluster assignment variable $z_i$ drawn from a discrete uniform distribution\+:   \[P(z_i = k) = \frac{1}{K}, \quad k \in \{0, 1, \dots, K-1\}
\] where $K$ is the number of target clusters.

Implementation Algorithm The {\ttfamily \doxylink{classcmp_1_1cluster_1_1_dummy_cluster_ad466acb58ffc955424660ef4da57089d}{fit()}} function constructs a {\ttfamily std\+::uniform\+\_\+int\+\_\+distribution} and samples an integer in the range $[0, K-1]$ for each data point sequentially.
