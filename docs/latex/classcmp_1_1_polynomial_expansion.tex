\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Basis$>$\newline
class cmp\+::\+Polynomial\+Expansion$<$ Basis $>$}
Implements multi-\/dimensional Polynomial Chaos Expansion (PCE) for spectral surrogate modeling. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Basis} & The orthogonal polynomial basis type (e.\+g., \doxylink{structcmp_1_1_hermite_basis}{Hermite\+Basis} for Gaussians, \doxylink{structcmp_1_1_legendre_basis}{Legendre\+Basis} for Uniforms).\\
\hline
\end{DoxyTemplParams}
\hypertarget{classcmp_1_1_polynomial_expansion_autotoc_md12}{}\doxysubsubsubsection{\texorpdfstring{Mathematical Foundations}{Mathematical Foundations}}\label{classcmp_1_1_polynomial_expansion_autotoc_md12}
Polynomial Chaos Expansion represents a stochastic model output $Y = f(\mathbf{X})$ as a spectral expansion of multivariate orthogonal polynomials mapping canonical variables $\boldsymbol{\xi}$\+: \[ Y \approx \sum_{j=0}^{P-1} c_j \Psi_j(\boldsymbol{\xi}) \]

The multivariate polynomials $\Psi_j(\boldsymbol{\xi})$ are tensor products of univariate orthogonal polynomials\+: \[ \Psi_j(\boldsymbol{\xi}) = \prod_{i=1}^d \psi_{\alpha_i^{(j)}}(\xi_i) \] where $\boldsymbol{\alpha}^{(j)} = (\alpha_1^{(j)}, \dots, \alpha_d^{(j)})$ is the multi-\/index of polynomial degrees.

Uni-\/dimensional orthogonal bases satisfy\+: \[ \int_{\Omega} \psi_k(x) \psi_l(x) w(x) dx = \gamma_k \delta_{kl} \] where $w(x)$ is the probability density function (PDF) weight of the canonical distribution.\hypertarget{classcmp_1_1_polynomial_expansion_autotoc_md13}{}\doxysubsubsubsection{\texorpdfstring{Implementation Algorithms}{Implementation Algorithms}}\label{classcmp_1_1_polynomial_expansion_autotoc_md13}

\begin{DoxyEnumerate}
\item {\bfseries{Canonical Mapping}}\+: Physical samples $\mathbf{x}$ are mapped to canonical space $\boldsymbol{\xi}$ using the basis-\/specific mappings ({\ttfamily map\+To\+Canonical}).
\item {\bfseries{Design Matrix Construction}}\+: Computes the regression matrix $\mathbf{A} \in \mathbb{R}^{N \times P}$ where\+: \[ A_{ij} = \Psi_j(\boldsymbol{\xi}^{(i)}) \]
\item {\bfseries{Ordinary Least Squares (OLS) Regression}}\+: The expansion coefficients $\mathbf{c}$ are found by solving the linear least-\/squares problem $\mathbf{A}\mathbf{c} \approx \mathbf{y}$\+: \[ \mathbf{c} = \left(\mathbf{A}^T \mathbf{A}\right)^{-1} \mathbf{A}^T \mathbf{y} \] We solve this system robustly using Column-\/\+Pivoted Householder QR decomposition ({\ttfamily Col\+Piv\+Householder\+QR}).
\end{DoxyEnumerate}\hypertarget{classcmp_1_1_polynomial_expansion_autotoc_md14}{}\doxysubsubsubsection{\texorpdfstring{Constraints \& Invariants}{Constraints \& Invariants}}\label{classcmp_1_1_polynomial_expansion_autotoc_md14}

\begin{DoxyItemize}
\item {\bfseries{Sample Size}}\+: The number of samples $N$ must satisfy $N > P$ (where $P = \text{multiIndex\_.size()}$) to avoid an underdetermined system and severe overfitting.
\item {\bfseries{Parameter Ranges}}\+: Bounding parameter vectors (e.\+g., mean/standard deviation or lower/upper bounds) must match the dimensionality of the input space. 
\end{DoxyItemize}
