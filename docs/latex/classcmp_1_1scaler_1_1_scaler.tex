\doxysubsection{Detailed Description}
Template virtual base class for feature scaling and transformation. 

{\bfseries{Mathematical}} {\bfseries{Formulation}} Define a transformation mapping $ T: \mathbb{R}^d \to \mathbb{R}^d $ and its inverse $ T^{-1} $\+: \[ y = T(x) \] \[ x = T^{-1}(y) \] Often represented linearly with intercept $ \mu $ and scaling factor/matrix $ S $\+: \[ T(x) = S^{-1}(x - \mu) \] \[ T^{-1}(y) = S y + \mu \]

{\bfseries{Implementation}} {\bfseries{Algorithm}} Pure virtual interface. Concrete implementations define the fitting, transforming, and inverse transforming steps.
