\doxysubsection{Detailed Description}
Standardizes features by removing the mean and scaling to unit variance using Cholesky decomposition. 


\begin{DoxyItemize}
\item 
\end{DoxyItemize}\hypertarget{classcmp_1_1scaler_1_1_standard_scaler_autotoc_md16}{}\doxysubsubsubsection{\texorpdfstring{Mathematical Formulation}{Mathematical Formulation}}\label{classcmp_1_1scaler_1_1_standard_scaler_autotoc_md16}
Given data matrix $ X \in \mathbb{R}^{n \times d} $, let $ \mu \in \mathbb{R}^d $ be the column-\/wise mean and $ \Sigma \in \mathbb{R}^{d \times d} $ be the covariance matrix. We decompose the covariance matrix via the lower Cholesky factor $ L $ such that\+: \[ \Sigma = L L^T \] The transformation maps\+: \[ y = L^{-1}(x - \mu) \] The inverse transformation maps\+: \[ x = L y + \mu \]
\begin{DoxyItemize}
\item \#\#\# Implementation Algorithm
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item {\bfseries{Fit}}\+: Calculate mean vector $ \mu $ and covariance matrix $ \Sigma $ of $ X $, then perform Cholesky LLT decomposition to obtain $ L $.
\item {\bfseries{Transform}}\+: Solve the lower triangular system $ L y = x - \mu $ to compute standardized data.
\item {\bfseries{Inverse Transform}}\+: Compute $ L y + \mu $ via matrix multiplication and vector addition. 
\end{DoxyEnumerate}
