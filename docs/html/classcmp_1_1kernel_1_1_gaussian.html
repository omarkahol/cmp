Detailed Description

Standard Gaussian density kernel.

Mathematical Formulation The standard multivariate Gaussian density kernel:

\[K(\mathbf{z}) = (2\pi)^{-D/2} \exp\left(-\frac{1}{2} \|\mathbf{z}\|_2^2 \right) \]
