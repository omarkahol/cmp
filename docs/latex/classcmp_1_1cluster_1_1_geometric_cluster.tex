\doxysubsection{Detailed Description}
Implements a standard k-\/means clustering algorithm. 

Mathematical Formulation K-\/means partitions $N$ observations $\{\mathbf{x}_1, \dots, \mathbf{x}_N\}$ where $\mathbf{x}_j \in \mathbb{R}^d$ into $K$ clusters $\mathbf{S} = \{S_1, \dots, S_K\}$ to minimize the within-\/cluster sum of squares (WCSS)\+:   \[\arg\min_{\mathbf{S}} \sum_{i=1}^K \sum_{\mathbf{x} \in S_i} \|\mathbf{x} - \boldsymbol{\mu}_i\|_2^2
\] where $\boldsymbol{\mu}_i$ is the centroid of the points in $S_i$.

Implementation Algorithm
\begin{DoxyEnumerate}
\item Initialize centroids $\boldsymbol{\mu}_i$ by randomly selecting $K$ distinct data points.
\item Assign each point $\mathbf{x}_j$ to the cluster with the nearest centroid\+:   \[   S_i^{(t)} = \left\{ \mathbf{x}_j : \|\mathbf{x}_j - \boldsymbol{\mu}_i^{(t)}\|_2 \le \|\mathbf{x}_j - \boldsymbol{\mu}_m^{(t)}\|_2 \ \forall m=1,\dots,K \right\}
\]
\item Update the centroids to be the arithmetic mean of all points in that cluster\+:   \[   \boldsymbol{\mu}_i^{(t+1)} = \frac{1}{|S_i^{(t)}|} \sum_{\mathbf{x}_j \in S_i^{(t)}} \mathbf{x}_j
\]
\item Iterate steps 2-\/3 until convergence (centroids no longer change or {\ttfamily max\+\_\+iter} is reached). 
\end{DoxyEnumerate}
