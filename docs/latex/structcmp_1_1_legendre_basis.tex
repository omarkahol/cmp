\doxysubsection{Detailed Description}
Legendre orthogonal polynomial basis. 

Mathematical Formulation Legendre polynomials $P_n(\xi)$ are orthogonal with respect to the uniform density function $w(\xi) = 1/2$ on $[-1, 1]$. The recurrence relation is\+:   \[(n+1) P_{n+1}(\xi) = (2n+1) \xi P_n(\xi) - n P_{n-1}(\xi)
\] with $\int_{-1}^1 P_m(\xi) P_n(\xi) d\xi = \frac{2}{2n+1} \delta_{mn}$.

Implementation Algorithm
\begin{DoxyEnumerate}
\item {\ttfamily \doxylink{structcmp_1_1_legendre_basis_a367a40281029852ebebe93e1bb038ed0}{evaluate()}} computes polynomial values using Bonnet\textquotesingle{}s recurrence relation.
\item {\ttfamily \doxylink{structcmp_1_1_legendre_basis_a8e98d75858c0f97f1ad2bace3dbac833}{get\+Jacobi\+Matrix()}} constructs the symmetric tridiagonal Jacobi matrix $\mathbf{J}$ with $\alpha_i = 0$ and off-\/diagonal $\beta_i = \frac{i}{\sqrt{(2i-1)(2i+1)}}$. 
\end{DoxyEnumerate}
