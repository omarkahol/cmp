\doxysubsection{Detailed Description}
Abstract base class for multivariate KDE bandwidth matrices. 

Mathematical Formulation The bandwidth matrix $\mathbf{H} \in \mathbb{R}^{D \times D}$ acts as a linear transformation scaling spatial distances\+:   \[\mathbf{z} = \mathbf{H}^{-1} (\mathbf{x}_1 - \mathbf{x}_2)
\] The smoothing kernel scales inversely with the determinant\+: $\frac{1}{\det(\mathbf{H})}$.

Implementation Algorithm Defines virtual methods to compute coordinate scaling ({\ttfamily apply}), matrix determinant ({\ttfamily determinant}), and parameter gradients for NLopt optimization.
