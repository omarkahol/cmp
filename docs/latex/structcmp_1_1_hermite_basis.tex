\doxysubsection{Detailed Description}
Hermite orthogonal polynomial basis. 

Mathematical Formulation Probabilist\textquotesingle{}s Hermite polynomials $H_n(\xi)$ are orthogonal with respect to the standard Gaussian density function $w(\xi) = \frac{1}{\sqrt{2\pi}} e^{-\xi^2/2}$. The recurrence relation is\+:   \[H_0(\xi) = 1, \quad H_1(\xi) = \xi, \quad H_{n+1}(\xi) = \xi H_n(\xi) - n H_{n-1}(\xi)
\] with $\mathbb{E}[H_m(\xi) H_n(\xi)] = n! \delta_{mn}$.

Implementation Algorithm
\begin{DoxyEnumerate}
\item {\ttfamily \doxylink{structcmp_1_1_hermite_basis_a9dc704a947055ad82d002f2ddb158f10}{evaluate()}} computes polynomial values using the 3-\/term recurrence relation.
\item {\ttfamily \doxylink{structcmp_1_1_hermite_basis_a5f49b695439eec44c7871830c50c0b50}{get\+Jacobi\+Matrix()}} constructs the symmetric tridiagonal Jacobi matrix $\mathbf{J}$ with diagonal $\alpha_i = 0$ and off-\/diagonal $\beta_i = \sqrt{i}$ to evaluate quadrature nodes via spectral decomposition. 
\end{DoxyEnumerate}
