\doxysubsection{Detailed Description}
Diagonal bandwidth matrix (independent parameter per dimension). 

Mathematical Formulation Represents a diagonal matrix $\mathbf{H} = \text{diag}(h_1, \dots, h_D)$. Coordinates are scaled independently\+:   \[z_d = \frac{x_{1,d} - x_{2,d}}{h_d}
\] The determinant is $\det(\mathbf{H}) = \prod_{d=1}^D h_d$.
